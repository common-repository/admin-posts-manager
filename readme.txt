An advanced/beautiful posts manager for WordPress.

== Description ==

Admin posts manager lets you manage your post in a beautiful and easy way with advanced filtering options right into your admin panel. No refresh page needed, you can view,edit your posts in real time.

### Features

 - [x] **View Posts** - Can view posts in real time.
 - [x] **Edit Posts** - Can quickly edit your posts.

 and many more coming soon...

### Easy To Configure

There are no extra configurations required, you just need to activate the plugin.

### Easy To Use Interface

It has a simple, user-friendly interface, through which you can manage your posts easily.

### Light Weight and Instant Loading

Code is optimized for fast loading and instant live editing with no extra resources.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress
